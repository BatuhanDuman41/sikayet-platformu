# Şikayet Platformu

Next.js + TypeScript + Tailwind CSS ile hazırlanmış şikayet, değerlendirme ve müşteri deneyimi platformu başlangıç projesi.

## Yerelde çalıştırma

```bash
npm install
npm run dev
```

## Vercel deploy

1. Bu projeyi GitHub repository içine yükleyin.
2. Vercel'de `New Project` seçin.
3. GitHub repository'yi import edin.
4. Framework olarak `Next.js` seçili kalsın.
5. `Deploy` butonuna basın.
